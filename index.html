Cloud Database Setup

Your products, orders, and reviews are stored in Firebase Realtime Database so they sync across all devices. To set up your own database:

  1. Go to firebase.google.com and create a free account
  2. Create a new project called "Nobel Wear"
  3. Go to "Build" → "Realtime Database" → "Create Database"
  4. Choose location asia-southeast1 (Singapore) for best Pakistan speed
  5. Set rules to: "read": true, "write": true (for now)
  6. Copy your database URL and paste it in the code at the top of <script> tag
